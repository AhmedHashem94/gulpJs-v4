Gulp v4



1 => install nodejs to have access to npm
2 => now install gulp cli globally run this command  =>   npm install -g gulp-cli 
3 => now install gulp locally with running this command =>  npm i --save-dev gulp 
4 => install all the packedges that we need by running this command  =>  npm i --save-dev gulp-sass gulp-notify gulp-autoprefixer gulp-csso gulp-concat browser-sync
